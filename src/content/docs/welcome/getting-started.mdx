FlashServer is a full-featured library build on top of EntityFramework Core and Automapper to help
developers build basic monolith servers for mini projects. this guide will help you started with a new server.

## Why FlashServer?

<CardGrid>
  <Card title="Easy To Develop" icon="rocket">
    With FlashServer creating endpoints are as easy as clicking a button, all you need are your data classes
  </Card>

  <Card title="Easy To Learning" icon="pencil">
    Because FlashServer is a wrapper library learning how to use the library is just one more step from EntityFramework Core
  </Card>

  <Card title="Easy To Modify" icon="setting">
    If your req are a bit different from the prefix there is a easy way to answer your needs,
    just write your need over the function and the library with include it automatic
  </Card>

  <Card title="Easy To Add" icon="add-document">
    If you need to add a new endpoint/model its as easy as creating a step each layer
  </Card>

</CardGrid>
